# 🧠 OPIUM and CitySpace Project Overview
The OPIUM and CitySpace project is a collection of web development projects that provide smooth scrolling functionality and responsive navigation menus for two different websites: OPIUM and CitySpace. The project includes two main JavaScript files, `script.js` and `main.js`, which handle the smooth scrolling and navigation menu functionality for the OPIUM and CitySpace websites, respectively. The project also includes an HTML file, `index.html`, which serves as the main entry point for the OPIUM website.

## 🚀 Key Features
* Smooth scrolling functionality to specific sections of the webpage
* Responsive navigation menu that toggles on and off when the burger icon is clicked
* Menu link click handler that closes the navigation menu when a link is clicked on smaller screens
* Easing function to calculate the easing of the scroll animation, providing a more natural scrolling experience
* Use of native JavaScript and DOM APIs to minimize dependencies

## 🛠️ Tech Stack
* Frontend: HTML, CSS, JavaScript
